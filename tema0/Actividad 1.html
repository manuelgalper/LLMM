HTML

Introducción

HyperText Markup Language es el lenguaje de marcado predominante para la elaboración de páginas web. Fue creado por Tim Berners-Lee en 1991.

Primeras especificaciones

La primera descripción pública de HTML fue un documento llamado "HTML Tags", publicado por Tim Berners-Lee. Las especificaciones iniciales fueron la base para el desarrollo de los estándares web actuales.
